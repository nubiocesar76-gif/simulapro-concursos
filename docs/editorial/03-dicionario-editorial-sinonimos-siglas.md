# 03 — Dicionário Editorial Mestre: Sinônimos, Palavras-chave e Siglas

Este arquivo é o **índice de busca rápida** usado pelo Motor Editorial para
fazer *matching* automático de texto de prova → disciplina/assunto. Os
detalhes completos de cada termo estão nos dossiês `02a`–`02l`; aqui a
informação é consolidada e cruzada, com foco especial nas **siglas
ambíguas** (mesma sigla, significados diferentes conforme a disciplina).

## 4. Regra geral de manutenção deste dicionário

Sempre que uma nova prova revisada (`docs/work/<prova>/review.json`) trouxer
um termo ambíguo não catalogado aqui, ele deve ser adicionado **antes** de a
questão ser publicada, para que o critério de classificação fique registrado
e reutilizável — nunca decidir uma ambiguidade "no calor da revisão" sem
documentar a regra aqui e/ou em `05-casos-ambiguos-regras-classificacao.md`.
